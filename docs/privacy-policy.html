X Bookmark Summarizer Privacy Policy

Last updated: March 25, 2026

X Bookmark Summarizer is a browser extension that helps users collect posts from their own X bookmarks page and summarize them using the user's OpenAI API key.

Data We Handle

The extension may handle your OpenAI API key, bookmark content visible on your X bookmarks page, bookmark metadata such as post text, author name, permalink, timestamp, extracted external links, and media references, your custom prompt and model settings, and the most recent generated summary.

How Data Is Used

The extension uses this data only to collect bookmarked posts from your logged-in bookmarks page, generate a summary when you explicitly request one, and store your local settings and last results inside Chrome extension storage.

Where Data Is Stored

The extension stores the following locally in Chrome extension local storage: your OpenAI API key, model and prompt settings, last collected bookmarks, and last generated summary.

Data Sent Off-Device

When you click Collect + Summarize, the extension sends collected bookmark data and prompt text to OpenAI through the Chat Completions API.

The extension does not operate its own backend for bookmark processing.

Data Sharing

The extension does not sell your data and does not share your data with advertising networks or analytics providers. The only third-party data transfer performed by the extension is the user-initiated request to OpenAI for summarization.

Data Retention

Data remains in your local extension storage until you remove it, clear extension storage, or uninstall the extension.

Security

The OpenAI API key is stored locally in Chrome extension storage. As with any browser extension, users should review the code and use the extension only on trusted machines and browser profiles.
