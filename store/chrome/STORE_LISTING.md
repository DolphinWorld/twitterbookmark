## Detailed Description

`X Bookmark Summarizer` helps you review saved X bookmarks faster.

What it does:
- Reads posts from your logged-in X bookmarks page
- Collects bookmark text, links, timestamps, and media references visible in the page
- Sends the collected bookmark data to OpenAI only when you click `Collect + Summarize`
- Shows the resulting summary directly in the extension popup
- Lets you export the last collected bookmark set as JSON

What it does not do:
- It does not use the official X API
- It does not run on every site
- It does not use its own backend
- It does not send data anywhere except OpenAI when you trigger summarization

Suggested closing note:
- `This is an unofficial utility for working with your own bookmarked posts while signed in to X.`
