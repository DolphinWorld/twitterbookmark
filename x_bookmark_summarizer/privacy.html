Privacy Summary

This page explains what the extension stores and when data is sent off-device.

What is stored locally

  • Your OpenAI API key
  • Your preferred model and system prompt
  • The last collected bookmark set
  • The most recent generated summary

What is sent to OpenAI

When you click Collect + Summarize, the extension sends the collected bookmark text, URLs, timestamps, and extracted links to the OpenAI Chat Completions API using the API key you provided.

What is not sent anywhere else

  • No analytics or tracking service is used
  • No bookmark data is sent to a separate backend owned by this extension
  • No advertising or sale of user data is involved

Scope

The extension only runs on the X/Twitter bookmarks page and when calling the OpenAI API for summarization.
